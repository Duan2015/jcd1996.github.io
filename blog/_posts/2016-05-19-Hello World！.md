# jekyll相关使用
顺便记录一下jekyll相关使用，以便之后需要


执行

```shell
    bundle install
    bundle exec jekyll s
```

  
启动本地web服务器，用浏览器打开localhost:4000即可本地预览

在`_post`文件夹下添加相同格式的md文件，每个md文件是一篇文章的内容
